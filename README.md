# Employee Management Microservice
** Hackathon practice demo 2026**

## Tech Stack
- Java 21 & Spring Boot 4.x
- Spring Data JPA (H2 In-Memory Database)
- GitHub for Version Control

## How to Run
1. Clone the repo.
2. Open in Spring Tool Suite 4/5.
3. Run as 'Spring Boot App'.
4. Access H2 Console at `http://localhost:8081/h2-console`.
